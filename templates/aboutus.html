About Our Ransomware Detection System

Understand the mission, methodology, and models behind our threat classification engine.

About Our System

Our project, “A System for Analyzing and Categorizing Ransomware Threats using Machine Learning Techniques,” aims to detect and classify ransomware variants to reduce financial and operational losses. It identifies threats like LockBit, BlackCat (ALPHV), Rhysida, Akira, and Play Ransomware by analyzing indicators such as domain entropy, network behavior, and encryption patterns.

🎯 Goal

To provide accurate and real-time classification of ransomware threats using intelligent ML models.

🧪 Approach

We use Random Forest, Support Vector Machine (SVM), and Logistic Regression models, with ANOVA F-statistic for feature selection.

📊 Evaluation

Models are evaluated based on accuracy, precision, recall, and F1-score for effective multi-class classification.

🔍 Features We Analyze

✔️ Domain Entropy
✔️ Vowel Ratio
✔️ Domain Length
✔️ Outbound Connections
✔️ DNS Query Rate
✔️ Exfiltration Indicator
✔️ Distinct IPs
✔️ Packet Size
✔️ Communication Time
✔️ Protocol Type
✔️ Non-Standard Ports
✔️ TLS Validity
✔️ User Agent
